## Online Medical Store (Medstore)

This is a web-based Online Medical Store project developed using PHP, HTML, CSS. Here user can register and login serach for pharmacies in their locality and add medicines to cart and buy them through online payment. If the medicine was high risk then the pharmacy will verify them and added that to corresponding users cart.The medical store will assign corresponding deliveryboy here and they will delivers the medicines to corresponding delivery address.

##  Features
- User login system (Admin, Users, Pharmacy, Delivery Boy, Guest)
- Medicine browsing and ordering system
- AJAX-based cart and dynamic updates
- Role-based access control

##  Tech Stack
- PHP(backend)
- HTML
- CSS
- MySQL (database)
